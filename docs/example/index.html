Example with JSON config fileâ€‹

Input file: test.txt

2021

Config file: config.json

{
    "lang": "en-CA",
    "input": "test.txt",
    "output": "./newDist",
    "stylesheet": "https://cdn.jsdelivr.net/npm/water.css@2/out/water.css"
}

Command ran

ssg -c config.json
